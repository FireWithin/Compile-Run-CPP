C++ Compile and Run Tool for Linux

About: 
Now compile and execute CPP files from anywhere your computer with ease.
GUI for this application is written using PyQt.

Shortcuts:
Run: 		Ctrl + 'R'
Compile: 	Ctrl + 'B'
Browse:		Ctrl + 'O'
Quit: 		Ctrl + 'Q'
